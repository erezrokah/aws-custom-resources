## Prerequisites

- [Install Node.js LTS](https://nodejs.org/en/)
- [Set up the Serverless Framework CLI](https://www.serverless.com/framework/docs/getting-started)

## Setup

```bash
npm ci
```

## Deploy

```bash
npm run deploy
```

## Cleanup

```bash
npm run remove
```

## Test

```bash
npm test
npm run coverage
npm run lint
npm run format
```

## Build

```bash
npm run package
```
